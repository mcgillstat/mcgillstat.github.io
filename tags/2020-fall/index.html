    BdryGP: a boundary-integrated Gaussian process model for computer code emulation

    Simon Mak · Sep 18, 2020

    Date: 2020-09-18 Time: 15:30-16:30 Zoom Link Meeting ID: 965 2536 7383 Passcode: 421254 Abstract: With advances in mathematical modeling and computational methods, complex phenomena (e.g., universe formations, rocket propulsion) can now be reliably simulated via computer code. This code solves a complicated system of equations representing the underlying science of the problem. Such simulations can be very time-intensive, requiring months of computation for a single run. Gaussian processes (GPs) are widely used as predictive models for “emulating” this expensive computer code.
